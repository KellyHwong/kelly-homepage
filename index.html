Realization of Micro Drone System Design and Application

系统硬件方面采用STM32F407DISCOVERY 开发板为核心处理器。该处理器内核架构为ARM-Cortex-M4,具有高性能、低成本、低功耗等特点。其他硬件模块用有、无线蓝牙模块、MPU6050 运动数据获取模块、超声波模块、无刷电机调节器（4 个）、无刷电机（4 个）。 软件控制方面。1.利用获取到的XYZ 三轴的加速度，以及系统与XYZ 三轴之间的夹角，采用四元素解算算法，获得系统当前的运动姿态，即ROL(翻滚角)、PIT(俯仰角)、YAW（方向角）。2.采用PID 自稳控制算法，使系统稳定的达到相对稳定的平衡状态。

The pity is that, due to lack of experience in PID arguments modifying, the micro drone didn’t work totally fine in the end. Nevertheless, I still got a lot in this project.

My Experience in MCLab

MCLab, which’s full name is Media and Communication Lab, is a producing laboratory located in HUST(Huazhong Univ. of Sci. and Tech.). During my internship in MCLab, I mainly research in the improvement in 3D model retrieval.

Basics of 3D model retrieval

There are two 3D model database, the larger one is called TSB and the smaller one is PSB. Using the extracting 64 pictures of each model of TSB as training data, train a neural network model. Extract features of each model in PSB. For each model of TSB and PSB, it has 64 feature vectors. Then calculate the Hausdorff distance between models. Predicting the type of a 3D model according to this distance.

Compare between torch framework and caffe

Caffe is a great framework, which is integrated with powerful functions, the only thing you need to do is to write prototxt configure files. But, the realization of its function is transparent for users, and moreover, different version of caffe may confront with each other, leading to a situation that merging all kinds of new function into it is a quite difficult project, unless you read through caffe’s source code. In the mean time, torch can guide you through the programming from modeling to neural network forward and backward computing. You can learn a lot from this. Also, torch is flexible enough to include and integrate all kinds of functions.

UAV Visual Navigation System Design Based on 2D Code

This is my graduation project. In this project, I mainly realize the recognition algorithm for a kind of 2D code called AprilTag code. This code is highly error detectable, easily to be recognized by a camera. It is better to be used as object tracking than QR code.

I learnt the programming of ROS(Robotic Operation System) on the DJI M100 platform, and transplant the algorithm into M100. After programming for the camera of M100, I realize the real-time AprilTag detection of the camera of M100.
